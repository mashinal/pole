# Smoke-тесты раздела "Урожай"

Проект содержит простые UI smoke-тесты публичного раздела "Урожай" на сайте поле.рф.

Стек: Java, Maven, Selenide, JUnit 5, Chrome.

## Что нужно перед запуском

1. Должна быть установлена Java 21.
2. Должен быть установлен Maven.
3. Должен быть установлен Chrome.

ChromeDriver отдельно скачивать не нужно. Его подбирает и запускает WebDriverManager.

## Как запустить тесты через терминал

Открой терминал и перейди в папку проекта:

Запусти тесты:

```bash
mvn clean test
```

По умолчанию браузер открывается в обычном режиме.

Для запуска в обычном режиме:

```bash
mvn test -Dheadless=false
```

Для запуска в headless:

```bash
mvn test -Dheadless=true
```

## Архитектура проекта

- `BaseTest` настраивает Chrome, WebDriverManager, таймауты и открывает главную страницу.
- `HomePage` описывает главную страницу.
- `HarvestPage` описывает страницу "Урожай": локаторы, фильтр, поиск и проверки карточек.
- `HarvestSmokeTest` содержит сами тесты.
